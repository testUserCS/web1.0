Description

The Moen Minecraft server uses an engineering and technology focused forge-based modpack. Version 1.0 was based on Feed The Beast's Direwolf20 pack, with some additional mods added. The latest version of the Moen modpack uses FTB Lite 3 as a base, to provide a better experience for users with less powerful computers.

Included Mods

Mods can be added by request. Contact the server admin at abelincoln.white@gmail.com to request an addition.
